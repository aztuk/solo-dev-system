# Flows Catalog

Index de tous les flows du design system. Géré par `figma-sync.md`.

Chaque flow représente un parcours utilisateur cohérent. Les pages individuelles sont documentées dans le fichier du flow correspondant.

## Légende

**Statut** : `draft` (en cours de design) | `stable` (validé, prêt à implémenter) | `implemented` (codé et validé)
**Source** : toujours `figma` — les flows sont définis dans Figma, jamais en dehors.
